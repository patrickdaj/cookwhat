# Ready-to-paste prompts

Copy any of these into Claude (with this repo open). Claude already knows the
workflow from `CLAUDE.md`, so a short ask is enough — these just give you good
starting points.

## Build next week's menu
```
Build me a balanced dinner menu for the week of <DATE>.
Use my cookwhat.config.json rules, pull real recipes from my preferred sites,
include one of my top redos, and keep weeknights quick. Save it and run the
rule check, then show me the plan with links.
```

## Build around what's in season / a craving
```
Plan this week's dinners with a Mediterranean lean and lots of seasonal
produce. Two vegetarian nights, one fish night, red meat only once. Save and
check it.
```

## Quick week
```
I'm slammed this week — every weeknight dinner under 25 minutes hands-on,
minimal cleanup, nothing fancy on the weekend either. Build and set it.
```

## Adjust a draft
```
Swap Wednesday for a Thai dish, make Friday vegetarian, and find a faster
Tuesday recipe. Re-check the plan.
```

## Make the shopping list
```
Set this week's menu and generate the shopping list, scaled for 6 people since
we have guests Saturday.
```

## Log how it went
```
Quick ratings: Thai basil chicken 5 stars and a definite redo; the pasta was a
3, too bland; short ribs 5 stars. Log these.
```

## Plan from my redos
```
Show my top redos, then build a "greatest hits" week mostly from meals we've
loved before, with one new recipe to try.
```
